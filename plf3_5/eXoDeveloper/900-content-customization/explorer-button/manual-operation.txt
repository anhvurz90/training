Manual operation

-- Modify the CSS file to add the button --
webapps/ecmadmin/skin/icons/24x24/DefaultStylesheet.css
...
.MyActionNameIcon {
	height: 24px;
  background: url('DefaultSkin/AddDocument.gif') no-repeat left center; /* orientation=lt */
	background: url('DefaultSkin/AddDocument.gif') no-repeat right center; /* orientation=rt */
}
....




--- Add the resources bundle properties ---

Add the name of the button for the ECM Admin portlet in the file
webapps/ecmadmin/WEB-INF/classes/locale/portlet/administration/ECMAdminPortlet_en.xml 

...
 <UIViewFormTabPane>
...
    <label>
      <myActionName>My Action</myActionName>
      <version>Restore Version</version>
....


--- Add the resource bundle for the File Explorer ----
Add the name of the button for the new button in the file 
webapps/ecmexplorer/WEB-INF/classes/locale/portlet/explorer/JCRExplorerPortlet_en.xml

...
  <UIActionBar>
    <tooltip>
      <MyActionName>My Action</MyActionName>
      <AddFolder>Add Folder</AddFolder>
....


=>> restart your server.
